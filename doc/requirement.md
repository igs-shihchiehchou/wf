這是一個音效處裡的靜態網站

功能:
1. 提供讀取 local 音效檔案
    - 讀取後產生一個編輯框(card)
    - 其中有一個播放/暫停的 toggle 按鈕
    - 可以拖拉撥放位置
    - 右方有下載按鈕

2. 每一個編輯框包含:
 - 輸入音效為上一個編輯框
 - 裁切功能: 透過 UI 裁切音效
 - 調整音量功能
 - 可分別加入淡入或淡出功能
 - 簡易的拉伸音效
 - 右下角一個執行按鈕，執行完後會編輯音效並向下產生新的編輯框

3. 此網站需透過 github page 部屬


開發需求:
每個項目階段皆須要透過 git 做版控，並且 commit pattern: {type}: {one line message}
